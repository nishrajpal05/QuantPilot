## Important

This project uses Render for backend hosting. Since the backend may go to sleep after inactivity, the first API request can take some time to respond.

While evaluating the project, if registration does not work on the first try, please retry 2–3 times after a few seconds. Once the backend wakes up, registration and all other functionalities work successfully.
